# Legacy research assets

These files preserve superseded experiments and implementation notes. They are
not part of the current name-only hierarchical ACSP-Discover workflow and are
not run by the default test suite.

## Contents

- `benchmarks/`: the original Izu four-island candidate benchmark and the first separate SDM model-accuracy benchmark.
- `benchmark_results/`: compact, tracked summaries from those benchmarks and the two pre-hierarchy national cohorts. Large per-fold candidate checkpoints remain gitignored.
- `tests/`: archived benchmark-specific tests, renamed so normal test discovery does not execute them.
- `scripts/`: one-off automatic-discover validation runners.
- `docs/`: the superseded Izu prospective field protocol.
- `notes/`: completed route-QC and temporary implementation notes.

Historical paths in `CHANGELOG_AI.md` describe where files lived when work was
performed. Use this index for their current locations.
